用hexo+GitHub搭建个人博客

2017-09-29

hexo

A fast, simple & powerful blog framework
一个快速简单强大的博客框架

github pages

为您和您的项目提供网站。
直接从您的GitHub信息库托管。只需编辑，推送和更改即可

hexo的特点

  1. 快速生成
    基于node.js,数百个文件几秒钟就能生成
  2. 支持markdown
    支持GitHub Flavored Markdown标准
  3. 单命令部署
    一个 hexo d 命令就可以将代码发布到GitHub Pages，Heroku或其他站点。
  4. 各种插件
    有强大的插件系统

搭建步骤

配置环境

  1. node.js
  2. git

安装hexo

生成页面

启动本地服务

注册github 并创建仓库

仓库名称必须以github用户名为前缀： username.github.io

修改配置文件

本地项目里有一个_config.yml文件，翻到最下面

deploy:
    type: git

    repo: git 地址

    branch: 分支名称

安装git插件

发布blog

至此就可以访问blog了.

域名是 username.github.io

一些常用命令

配置自己的域名

  1. 首先注册一个域名
  2. 通过CNAME类型解析到github pages的地址
  3. 在项目中的source文件夹下建立CNAME文件，写上你的域名即可
