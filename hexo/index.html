hexo

A fast, simple & powerful blog framework
一个快速简单强大的博客框架

github pages

为您和您的项目提供网站。
直接从您的GitHub信息库托管。只需编辑，推送和更改即可

hexo的特点

  1. 快速生成

    基于node.js,数百个文件几秒钟就能生成

  2. 支持markdown

    支持GitHub Flavored Markdown标准

  3. 单命令部署

    一个 hexo d 命令就可以将代码发布到GitHub Pages，Heroku或其他站点。

  4. 各种插件

    有强大的插件系统
